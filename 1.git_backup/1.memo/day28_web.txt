2. flex box(Flexible Box Layout) ★★★★★
1) layout을 만들 때 사용하는 속성
	float
		예전방식
		주로 이미지 옆에 글 배치할 때 사용
		레이아웃 구성시에는 불편함이 많음

	display의 flex box
		많이 사용하는 레이아웃 방식
		1차원 레이아웃(가로 또는 세로)
		정렬, 간격, 순서 제어가 쉬움

	grid
		2차원 layout(행 + 열)
		page 전체 구조에 적합
		flex보다 구조가 큼

2) flex box
	layout을 만드는 하나의 방식이다
	복잡하지 않게 유연하게 요소들의 크기와 순서를 정하여 배치할 수 있다

3) 구성
	flex container	: layout을 적용하여 배치하려는 요소들을 감싸고 있는 부모요소
			자식 요소들을 한 줄로 묶어 배치한다

	flex item	: 실질적으로 배치하려는 요소이며 flex container의 자식요소
			자식요소인 flex item은 여러개 만들 수 있다

	주측(main axis)	: 자식 요소가 배치되는 방향
		 	item이 나열되는 방향, 기본값은 가로 방향
			주축 기준으로 정렬하는 속성 : justify-content

	교차측(cross axis) : 주측을 가르지르는(수직인) 방향
			row일 경우 -> 세로
			column일 경우 -> 가로
			교차축 기준으로 정렬하는 속성 : align-items

4) flex Container 만들기
	컨테이너로 사용할 부모 요소에 display:flex 속성을 지정한다

5) flex Container 주요 속성
	배치하려는 요소들의 전체 흐름, 정렬과 관련된 속성

- display 
	display:flex	
		flex container를 지정하는 속성

	flex-direction:row|column;
		주측 방향을 지정하는 속성(기본값 : row)

	justify-content: center | space-between | space-around;
		container 주축을 기준으로 아이템 정렬
		
		center	중앙정렬
				[     ■  ■  ■    ]
		spece-between	첫 아이템과 끝 아이템을 컨테이너 양쪽에 배치하고 나머지 아이템들은 동일한 간격 정렬
				margin이 없으면 완전이 끝으로 붙는다(양끝 + 균등간격)
				[■       ■      ■]
		space-around	모든 아이템을 동일한 간격으로 정렬(모든요소 동일간격)
				[   ■   ■   ■   ]
		flex-start	시작점 정렬(기본값)	
				[■ ■ ■]

	align-iems: center | stretch | flex-start | flex-end
		교차축 기준의 item 정렬방식
	
		center		중앙정렬
	
		stretch		컨테이너 높이만큼 늘어남(기본값)

		flex-start	위쪽정렬

		flex-end	아래쪽 정렬
	
		baseline	텍스트 기준선 정렬

	flex-wrap: nowrap | wrap | wrap-reverse;
		item 줄바꿈 여부
	
		nowrap		한줄에 모두 배치(기본값)
	
		wrap		공간 부족시 자동줄바꿈

		wrap-reverse	줄바꿈 방향 반대로


3. display 속성의 기본 5가지
1) display
	HTML 요소를 화면에 어떻게 보여줄지 결정하는 CSS 속성

2) display 속성
- display:block
	한 줄 천체 차지
	자동 줄바꿈
	width/height 지정 가능

- display:inline
	내용만큼만 차지
	같은 줄 배치
	width/height 지정 불가능

- display:inline-block
	Inline 처럼 한 줄 배치
	block처럼 width/height 지정 가능

- display:none
	화면에서 완전히 제거
	공간도 차지하지 않음

- display:flex
	요수를 flex container로 만듦
	자식 요소가 flex item이 됨
	layout 배치 전용

5. position
	web site에서 특정 위치에 고정시켜서 화면을 스크롤해도 같은 위치에 있는 요소들이 있다
	어떤 위치에 고정시키거나 위치를 변경시킬 때 position 속성을 사용한다

1) position 속성
	HTML 요소를 배치하는 속성
	
	static
		position 속성의 기본값
		일반적인 흐름으로 배치된다

	relative
		static과 동힐하게 배치되지만 relative는 현재 자신의 위치를 기준으로 배치를 변경할 수 있다
		top, right, bottom, left 속성을 사용한다

	absolute
		일반적인 흐름과 무관하게 배치할 수 있다
		position:static이 아닌 부모요소를 기준으로 배치한다
		만약 조상 요소 중 static이 아닌 요소가 없다면 body를 기준으로 배치된다

	fixed 
		브라우저 화면상으로 보이는 영역(viewport)에 고정되도록 배치한다
		즉, 스크롤해도 고정된다

	sticky
		처음에는 일반적인 흐름에 배치되고 스트롤이 특정 위치에 도달하면 fixed처럼 화면에 고정된다
		주의사항 : 부모 요소에 height가 지정되어 있어야만 작동한다

2) z-index
	요소가 배치될 때 겹치게된다면 어떤 요소가 위로 올라올 지 순서를 정하는 속성
	기본값은 auto이며 지정하는 값은 1, 10, 100, -1, -100 등의 숫자를 사용한다
	숫자가 높을수록 요소가 앞으로 나오게 된다
	주의사항 : position이 static이 아닌 요소에만 적용된다!!

6. 고급 선택자
1) 연결 선택자
- 하위 선택자
	상위요소 하위요소{}
	상위 요소에 포함된 모든 하위 요소를 선택한다
	자식을 포함한 모든 자손이 선택된다

- 자식 선택자
	상위요소 > 하위요소{}
	부모요소(상위요소) 바로 아래에 속한 자식 요소를 선택한다

- 인접형제선택자
	요소1 + 요소2

- 형제선택자
	요소 ~ 요소2

2) 속성 선택자
- 선택자[속성]
- 선택자[속성=속성값]

3) 가상 선택자
- 가상 클래스
	선택자:가상클래스{}
		실제 존재하는 요소의 특정 상태에 따라 CSS를 주고 싶을 때 사용
	link	href 속성을 가진 태그 중 아직 방문하지 않은 상태
	visited	이미 방문한 상태
	hover	사용자가 마우스를 올린 상태
	focus	입력 가능한 칸을 선택한 상태
	nth-child	형제 중 순서로 선택한다
	not(선택자)	해당 선택자를 제외하고 선택한다

- 가상 요소
	선택자::가상요소{}
		요소의 특정부분에 스타일을 지정ㅇ하거나 새로운 요소를 삽입하기 위해 사용한다
	first-letter	첫글자만 선택(블록요소에서만 가능)
	first-line	첫 줄만 선택(
	before
