1. Javascript JSON 변환
1) JSON
	data를 주고받기 위한 text 형식
	server 클라이언트로 data를 보낼 때 Javascript 객체를 그래도 전송할 수 없음
	문자열로 변환해서 전송하고 받은 쪽에서 다시 객체로 복원하는 방식을 사용함

	[클라이언트]			[서버]
	Javascript 객체 -> (stringify) -> JSON 문자열 -> 전송
	Javascript 객체 <- (parse)     <- JSON 문자열 <- 수신
	
	JSON.stringify()	Javascript 객체를 JSON 문자열로 변환
				JSON 문자열을 Javascript 객체로 변환

2) JSON 문법 규칙
	javascript 객체(key에 따옴표 없어도 됨)
	
	const parson = {
		name : "홍길동",
		age : 20
	};

JSON 문자열(key도 반드시 큰 따옴표)
	'{"name":"홍길동", "age":"20"}'

- key 반드시 큰따옴표(작은따옴표 불가)
- 문자열값도 큰따옴표(작은따옴표 불가)
- 마지막 항목에 쉼표 불가
- 함수, undefined 불가(문자열, 숫자, 배열, 객체, boolean, null만 가능)

3) JSON.stringify()
	javascript 객체를 JSON 문자열 반환

4) JSON.parse()
	JSON 문자열을 javascript 객체로 변환

2. Javascript 총 복습
1) javascript

	HTML(구조) + CSS(디자인) + JS(동작)으로 web을 구성함

2) Data Type
- 원시 타입

	boolean	(true, false)

- 객체 타입
	object	(키-값 쌍 구조)

3) 변수선언

4) 연산자

10) 자바스크립트 엔진 구조

16) 동기/비동기
   javascript는 싱글 쓰레드지만 이벤트 루프 덕분에 비동기 처리가 가능하다
   
- Promise
   비동기 작업의 결과를 감싸는 객체
   3가지 상태
   - pending : 아직 완료되지 않은 상태
   - fulfilled : 성공한 상태(resolve)
   - rejected : 실패한 상태(reject)
   에러처리 .catch()

- async/await
   동기 코드처럼 직관적
   에러 처리 try/catch()

- fetch
   브라우저 내장 함수로 HTTP 요청을 보내고 Promise 반환

   get요청
   fetch("url")
   .then(response => response.json())
   .then(data => console.log(data))
   .catch(error => console.error(error));

   post요청
   fetch("url", {
      method:"post",
      headers :{"Content-Type":"application/json"},
      body:JSON.stringify({name:"홍길동"})
   });
